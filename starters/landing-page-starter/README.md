# 🚀 Landing Page Starter

`landing-page` workflow'undan türetilmiş Awwwards-kalitesinde landing page örneği.

## İçerik

- Next.js 14 + TypeScript + Tailwind
- GSAP ScrollTrigger sahne animasyonları
- Lenis smooth scroll
- Framer Motion hero animasyonu
- web-vitals raporlama
- Three.js + R3F decorative 3D objesi
- Lighthouse: Performance 95+, A11y 100, SEO 100

## Çalıştırma

```bash
pnpm install && pnpm dev
```

Detaylar: `orchestration/workflows/landing-page.json`
